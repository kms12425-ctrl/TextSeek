# search-engine-server

This is the Spring Boot backend module that exposes the core search capabilities through HTTP APIs.

## Main Responsibilities

- Expose search-related endpoints
- Handle document upload, deletion, and detail queries
- Manage index initialization and rebuilds
- Provide a unified response format
