# search-engine-server

这是 Spring Boot 后端模块，对核心搜索能力提供 HTTP API 封装。

## 主要职责

- 暴露搜索相关接口
- 处理文档上传、删除与详情查询
- 管理索引初始化与重建
- 提供统一响应结构
