# search-engine-core

这是搜索引擎的核心模块，负责倒排索引、解析处理、查询执行和结果组织。

## 主要职责

- 构建与维护索引
- 解析文档与查询内容
- 执行关键词检索
- 组织排序后的命中文档结果
